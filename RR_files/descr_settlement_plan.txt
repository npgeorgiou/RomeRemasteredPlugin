;
;  Settlement plans.
;
;  A 'plan set' specifies the full plan scheme used for a settlement, whatever its size
;  and culture.  For each culture and settlement level, there is a single 'plan level',
;  which determines where buildings can go, what ambients will be present, and what effect the
;  settlement has on the terrain.
;
;  To achieve this, a plan level has a list of ambients, and a list of main building slots.
;  Each slot consists of a variety of layouts, depending on size of major building being placed.
;  A slot can be restricted to a subset of placeable buildings, eg a slot may be reserved for
;  amphitheatres.  The types are listed in the building database.  It's done as a mask,
;  so can have multiple uses.
;
;  Walls will become part of this scheme as well, with limiting and preferred outlines for each plan level.
;
;  Positions are given as x, z, rotation (degrees anticlockwise), height offset (from plan or list origin)
;

; ie campaign-level buildings.  These will be used to determine damage to campaign buildings, and where to show key
; effects like indicative damage smoke.
